# 更新日志

## [2026-03-28 23:45:00]

🌈 本次更新重点：现在除了 uTools 自带的 AI，还能接上自己的 OpenAI 或 Claude，书签的标题简介和分类推荐都能用。同时 AI 助手从通用设置里独立出来，变成了单独的设置页，配置更清晰。

✨ 新功能
• 新功能：现在你可以在设置里切到"自定义 AI"，填上自己的 OpenAI 或 Claude 地址和密钥，模型列表会自动拉出来让你选
• 新功能：AI 助手有了专属设置页，开关、来源、模型选择都在一个地方搞定，不用再挤在通用设置里
• 新功能：书签现在支持 MCP 工具调用，AI 助手可以直接帮你查、搜、新建、打开书签，不用手动操作

🪐 优化
• 优化：分组和子分组选中时的样式换成了实心高亮，亮色暗色下都更容易看清当前选的是哪个
• 优化：新增书签时，AI 按钮不再要求必须在 uTools 环境才能点，配好了自定义 AI 也能用
• 优化：AI 调用出错时的提示更具体，会告诉你哪里配得不对

## [2026-02-28 20:08:00]
